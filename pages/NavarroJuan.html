PROYECTOS PERSONALES

DevSyntax Academy

Esta plataforma representa mi proyecto más ambicioso hasta la fecha. Fue desarrollada con React JS y diseñada con el objetivo de ofrecer una experiencia de aprendizaje moderna para quienes desean iniciarse en el mundo de la programación. El proyecto incorpora autenticación de usuarios mediante un sistema de Login y un backend implementado con Supabase. Actualmente la plataforma está en proceso de mantenimiento y mejora continua. Ya cuenta con profesores para cursos de Inglés para Programadores, HTML, CSS y JavaScript, y próximamente se ampliará con nuevos contenidos de React y Python.
